## Challenges Encountered and How I Solved Them:

 Challenge 1: Unable to Find Extensions in Cursor

**Issue**

After installing Cursor, I could not find the Extensions tab. I initially looked in the Customize section and was unable to locate Claude Code or Codex.

**Solution**

After researching and watching tutorial videos, I realized I had installed a different version of Cursor. I uninstalled it and installed the latest system version. Once installed correctly, the Extensions tab became available and I was able to install both Claude Code and Codex successfully.

---

 Challenge 2: Understanding How to Open a GitHub Repository in Cursor

**Issue**

This was my first experience working with repositories, Git, AI coding tools, and development environments. I found the process of connecting a GitHub repository to Cursor confusing and struggled to understand the workflow.

**Solution**

I watched multiple tutorials and experimented with different approaches. Through Cursor's AI assistant and additional research, I learned that Git was required to clone repositories locally. I installed Git for Windows, copied the repository clone URL from GitHub, cloned the repository into a local folder, and successfully opened it inside Cursor.

---

 Challenge 3: Installing Unnecessary Tools During the Learning Process

**Issue**

While learning how repositories work, I installed GitHub Desktop because I initially believed it was required to connect GitHub with Cursor.

**Solution**

After further exploration, I learned that Git for Windows was the essential requirement and that repositories could be cloned directly into Cursor using the repository URL. Although GitHub Desktop was not necessary for this task, the experience helped me better understand the GitHub ecosystem and repository workflows.
